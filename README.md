# Sailboat Vehicle Mod (帆船载具模组)

Minecraft Forge 1.20.1 多人帆船与国家系统模组。

## 功能介绍

### 帆船系统
- 可驾驶的多人帆船载具，支持5个座位和27格物品栏
- 帆/引擎双动力模式，基于节的速度物理系统
- 自动驾驶功能，支持航线跟随、停靠检测和区块加载
- 帆船可自定义名称和租赁价格

### 码头系统
- 可放置的码头方块，管理帆船停靠区域
- 支持航线规划和调度功能
- 码头间货物运输

### 市场系统
- 市场方块支持物品上架、购买和发货
- 与码头联动的物流运输系统
- 待领取货款管理

### 国家系统
- 创建和管理国家，设置国名、颜色、简称
- 职位体系：领袖、官员、成员，可自定义职位权限
- 邀请/申请加入机制
- 国旗上传和展示（支持镜像）

### 城镇系统
- 在国家下创建城镇，设置镇长
- 城镇核心方块标记城镇位置
- 城镇旗帜展示

### 领地圈地系统
- 基于区块的领地声明，支持国家和城镇级别
- 领地地图可视化，鼠标悬停显示领土信息
- 区域批量选择（左键选点 + Shift左键选区域）
- 7种权限控制：破坏、放置、使用、容器、红石、实体交互、实体伤害
- 6个权限等级：成员、官员、领袖、盟友、中立、所有人
- 领地保护：防爆、防火、防流体入侵

### 外交系统
- 国家间结盟、宣战
- 战争系统，支持实时战况追踪

### 国库系统
- 国库管理货币和物品存储（54格）
- 银行方块：可放置在城镇内，支持存取货币和物品
- 全息浮动名显示 "[城镇名]银行"

### 银行建筑器
- 手持银行建筑器时显示绿色线框预览建筑占地面积
- 放置后自动逐层建造维多利亚银行建筑（约10秒）

### 经济集成
- 可选 Vault 经济插件集成（通过反射，无硬依赖）
- 支持 Mohist/Catserver/Arclight 混合服务端

### BlueMap 集成
- 可选 BlueMap 地图标记集成
- 在网页地图上显示码头、帆船、航线、城镇/国家核心和国家边界

### 聊天集成
- 聊天前缀显示国家简称和职位
- 兼容 EssentialsX 插件（Bukkit 显示名同步）
- Tab列表显示国家信息

## 技术信息

- Minecraft 版本: 1.20.1
- Forge 版本: 47.2.0
- Java 版本: 17
- 依赖: GeckoLib（实体动画）

## 构建

```bash
./gradlew build
```

输出 JAR 在 `build/libs/` 目录下。
